Frequently Asked Questions

Is my data secure?

‚ñº

Yes! We use RSA 2048-bit asymmetric encryption, which is military-grade security. Your messages are encrypted on your device and can only be decrypted with the password you provide. We never store your passwords or plaintext messages.

Do you store my personal information?

‚ñº

No! This application does not require user accounts or store any personal information. We only temporarily store encrypted keys (not passwords or messages) for 3 days to enable decryption, after which they are automatically deleted.

What happens to my encrypted data?

‚ñº

We do NOT store your encrypted messages or plaintext data. You receive the encrypted data immediately and should save it locally. We only temporarily store the decryption key (encrypted with your password) for 3 days to enable decryption. After that, the key is automatically deleted. Without the key, your data remains secure but cannot be decrypted.

How long are my keys stored?

‚ñº

Encrypted keys are automatically deleted after 3 days using DynamoDB's TTL (Time To Live) feature. This ensures temporary storage without manual cleanup while maintaining security.

Can I use this for sensitive business data?

‚ñº

This tool provides strong encryption suitable for most personal and small business use cases. For highly sensitive or regulated data, consider dedicated encryption software with additional security features like key rotation and audit trails.

What encryption algorithm is used?

‚ñº

We use RSA 2048-bit asymmetric encryption for the data, combined with AES-256 for key protection. This hybrid approach provides both security and performance for web applications.

Is the source code available?

‚ñº

Yes! The complete source code is available on GitHub. This ensures transparency and allows security researchers to verify the encryption implementation.
